# AI Scholarship Recommendation Platform

## Project Type
Academic AI Web Application

## Stack
- NextJS (App Router)
- TailwindCSS
- shadcn/ui
- lucide-react
- framer-motion

## Goal
Membangun AI Agent yang membantu mahasiswa menemukan beasiswa yang sesuai dengan profil akademik dan kondisi pribadi mereka.

AI akan:
- memahami profil pengguna
- mencocokkan dengan database beasiswa
- memberikan top 3 rekomendasi
- menjelaskan alasan kecocokan

## Landing Page Goal

Landing page berfungsi untuk:

1. Menjelaskan masalah pencarian beasiswa
2. Menjelaskan solusi AI
3. Menjelaskan cara kerja sistem
4. Mendorong user untuk memulai chat dengan AI

Landing page harus terasa:

- akademis
- profesional
- informatif
- minimalis

Inspirasi UI:

- OpenAI
- trae.ai
- Vercel
- Notion AI
- attio.com
