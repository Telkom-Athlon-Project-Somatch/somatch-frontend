# Problem Section

## Tujuan

Menjelaskan pain points mahasiswa dalam mencari beasiswa.

User harus merasa relate dengan masalah yang ditampilkan.

## Layout

Grid 3 atau 4 card.

## Problem Points

1.

Informasi beasiswa tersebar di berbagai platform sehingga mahasiswa harus membuka banyak website.

2.

Banyak beasiswa tidak sesuai dengan kualifikasi seperti batas IPK atau persyaratan tertentu.

3.

Informasi persyaratan seringkali tidak konsisten antar sumber.

4.

Mahasiswa sering melewatkan deadline pendaftaran.

## Component

Gunakan:

Card component dari shadcn.

## Style

Minimalis dan akademis.
